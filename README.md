# Ridgeline Crane

Marketing website for **Ridgeline Crane** — a locally owned crane & rigging
service in Park City & Summit County, Utah.

Single-page static site (`index.html`) with a full-screen video hero, a
photographic bento work gallery, and LocalBusiness JSON-LD for local SEO.
Brand photos and video live in `reference/`.

## Local preview
    python -m http.server 4178
then open http://localhost:4178

## Deploy
Served via GitHub Pages from `main` (`.nojekyll` disables Jekyll processing).
